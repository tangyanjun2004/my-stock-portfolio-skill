# 股票投资组合 Skill - 需求文档

## 1. 项目概述

本项目是一个 Claude Code Skill，用于管理用户的股票持仓和关注股票列表，提供统一的 CLI 命令供 OpenClaw 使用。

## 2. 核心功能

### 2.1 持仓股票管理

| 功能 | 描述 |
|------|------|
| 添加持仓 | 添加新的持仓股票记录（股票代码、名称、数量、买入价、买入日期） |
| 更新持仓 | 修改已有持仓信息（数量、补仓/减仓记录） |
| 删除持仓 | 移除某支股票的持仓 |
| 查看持仓列表 | 显示所有持仓股票及其基本信息 |
| 查看单个持仓详情 | 显示某支股票的详细持仓信息 |

### 2.2 关注股票管理

| 功能 | 描述 |
|------|------|
| 添加关注 | 添加股票到关注列表（股票代码、名称、备注） |
| 移除关注 | 从关注列表中移除某支股票 |
| 查看关注列表 | 显示所有关注的股票 |

### 2.3 交易记录查询

| 功能 | 描述 |
|------|------|
| 查看所有交易记录 | 显示所有历史交易 |
| 按股票查询交易记录 | 显示某支股票的交易记录 |
| 按时间范围查询 | 显示指定时间段内的交易记录 |

### 2.4 账户管理与查询

| 功能 | 描述 |
|------|------|
| 初始化账户 | 创建账户，设置初始资金 |
| 查看账户概览 | 显示账户总额、可用金额、持仓市值、总投入、持仓数量等统计信息 |
| 更新账户信息 | 修改账户名称、转入/转出资金 |

### 2.5 数据持久化

- 使用 SQLite 数据库存储数据
- 支持数据备份（可选）

### 2.6 配置文件

| 功能 | 描述 |
|------|------|
| 配置文件 | 支持通过 JSON 配置文件自定义数据库路径 |
| 默认路径 | 不配置时使用默认路径 |
| 配置查看 | 支持查看配置的 CLI 命令 |

## 3. 数据库设计

### 3.1 表结构

#### 3.1.1 账户表 (accounts)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | INTEGER | 主键 | PRIMARY KEY AUTOINCREMENT |
| name | TEXT | 账户名称 | NOT NULL |
| total_balance | REAL | 账户总额 | NOT NULL DEFAULT 0 |
| available_balance | REAL | 可用金额 | NOT NULL DEFAULT 0 |
| created_at | TEXT | 创建时间 | NOT NULL |
| updated_at | TEXT | 更新时间 | NOT NULL |

#### 3.1.2 持仓表 (positions)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | INTEGER | 主键 | PRIMARY KEY AUTOINCREMENT |
| account_id | INTEGER | 账户ID | REFERENCES accounts(id) |
| stock_code | TEXT | 股票代码 | NOT NULL |
| stock_name | TEXT | 股票名称 | NOT NULL |
| quantity | REAL | 持有数量 | NOT NULL |
| average_cost | REAL | 平均成本价 | NOT NULL |
| created_at | TEXT | 创建时间 | NOT NULL |
| updated_at | TEXT | 更新时间 | NOT NULL |
| UNIQUE | | (account_id, stock_code) | |

#### 3.1.3 交易记录表 (transactions)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | INTEGER | 主键 | PRIMARY KEY AUTOINCREMENT |
| position_id | INTEGER | 持仓ID | REFERENCES positions(id) |
| type | TEXT | 交易类型 | CHECK(type IN ('buy', 'sell')) |
| stock_code | TEXT | 股票代码 | NOT NULL |
| stock_name | TEXT | 股票名称 | NOT NULL |
| quantity | REAL | 数量 | NOT NULL |
| price | REAL | 价格 | NOT NULL |
| amount | REAL | 交易金额 | NOT NULL |
| date | TEXT | 交易日期 | NOT NULL |
| note | TEXT | 备注 | |
| created_at | TEXT | 创建时间 | NOT NULL |

#### 3.1.4 资金变动记录表 (account_transactions)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | INTEGER | 主键 | PRIMARY KEY AUTOINCREMENT |
| account_id | INTEGER | 账户ID | REFERENCES accounts(id) |
| type | TEXT | 变动类型 | CHECK(type IN ('deposit', 'withdraw')) |
| amount | REAL | 金额 | NOT NULL |
| note | TEXT | 备注 | |
| created_at | TEXT | 创建时间 | NOT NULL |

#### 3.1.5 关注列表 (watchlist)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | INTEGER | 主键 | PRIMARY KEY AUTOINCREMENT |
| stock_code | TEXT | 股票代码 | NOT NULL UNIQUE |
| stock_name | TEXT | 股票名称 | NOT NULL |
| note | TEXT | 备注 | |
| created_at | TEXT | 创建时间 | NOT NULL |

### 3.2 ER 图示意

```
accounts (1) ──< (N) positions
accounts (1) ──< (N) account_transactions
positions (1) ──< (N) transactions
watchlist (独立表)
```

## 4. CLI 命令设计

### 4.1 持仓相关命令

```bash
# 添加持仓
my-stock-portfolio-cli position add --code <code> --name <name> --quantity <qty> --price <price> [--date <date>]

# 查看持仓列表
my-stock-portfolio-cli position list

# 查看单个持仓详情
my-stock-portfolio-cli position show <code|id>

# 更新持仓（买入/卖出）
my-stock-portfolio-cli position update <code|id> --type <buy|sell> --quantity <qty> --price <price> [--date <date]

# 删除持仓
my-stock-portfolio-cli position remove <code|id>
```

### 4.2 关注列表相关命令

```bash
# 添加关注
my-stock-portfolio-cli watch add --code <code> --name <name> [--note <note>]

# 查看关注列表
my-stock-portfolio-cli watch list

# 移除关注
my-stock-portfolio-cli watch remove <code|id>
```

### 4.3 交易记录相关命令

```bash
# 查看所有交易记录
my-stock-portfolio-cli transaction list [--stock <code>] [--start <date>] [--end <date>]
```

### 4.4 账户相关命令

```bash
# 初始化账户
my-stock-portfolio-cli account init --name <name> --balance <amount>

# 查看账户概览
my-stock-portfolio-cli account info

# 转入资金
my-stock-portfolio-cli account deposit <amount> [--note <note>]

# 转出资金
my-stock-portfolio-cli account withdraw <amount> [--note <note>]
```

### 4.5 配置相关命令

```bash
# 查看配置
my-stock-portfolio-cli config show
```

### 4.6 通用命令

```bash
# 显示帮助
my-stock-portfolio-cli help

# 显示版本
my-stock-portfolio-cli --version
```

## 5. 配置文件

### 5.1 配置文件格式

配置文件名为 `src_config.json`，与可执行文件放在同一目录下。

```json
{
    "db_path": "portfolio.db"
}
```

### 5.2 配置项说明

| 配置项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| db_path | string | 否 | 数据库文件或目录路径 |

### 5.3 db_path 配置规则

- 如果不配置，使用默认路径：
  - 开发阶段：项目目录下的 `data/portfolio.db`
  - 打包后：用户目录下的 `.src/portfolio.db`

- 如果配置为文件路径，直接使用该文件
- 如果配置为目录路径，会在该目录下创建 `portfolio.db`
- 支持相对路径和绝对路径

**注意**：配置文件需要手动创建和编辑，不提供 CLI 命令进行配置修改。
